---

### What I learned

In this project, I reinforced several core frontend concepts:

- How to structure a component using **semantic HTML** (`main`, `article`, `section`, `picture`).
- How to combine **Bootstrap utilities** with **custom CSS** to keep styles minimal and maintainable.
- How to implement **responsive images** correctly using the `<picture>` element instead of CSS hacks.
- How to follow a scalable **naming convention** inspired by BEM for custom styles.
- How to match a design system precisely using CSS variables from a style guide.

This approach allowed me to reduce custom CSS while still achieving a pixel-accurate result.
